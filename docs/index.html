The 10 Apps

real-engineer

General disciplined engineering router.

repo-cartographer

Map unfamiliar repos before changing them.

issue-sherpa

Turn fuzzy issues into scoped work.

pr-reviewer

Review diffs for bugs and regressions.

security-scout

Check auth, data, injection, deps, and config risks.

perf-lab

Optimize with measurements, not vibes.

release-captain

Prepare releases, notes, tags, and rollback checks.

docs-keeper

Keep docs aligned with shipped behavior.

frontend-polisher

Polish UI across viewports and states.

migration-pilot

Move APIs, deps, schemas, names, and files safely.

Why It Exists

Many public agent collections are large, overlapping, or tied to one tool's hooks and slash commands. This pack keeps the useful workflow ideas, narrows the triggers, and writes them as Codex-native skills.
