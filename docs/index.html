Real Engineer for Codex

A Codex skill for software work that needs evidence, feedback loops, and engineering judgment: diagnose, TDD, grill, prototype, and architecture review.

Star on GitHub Install the Codex Skill

One SKILL.md, Five Modes

Diagnose

For broken, flaky, slow, or failing behavior. Codex reproduces before guessing.

TDD

For explicit test-first work. Codex moves in red, green, refactor slices.

Grill

For fuzzy feature requests. Codex clarifies constraints before implementation.

Prototype

For uncertain UI, workflows, and state models. Codex creates disposable proof points.

Architecture

For refactors and maintainability. Codex studies coupling, boundaries, and interfaces.

Quick Install

curl -fsSL https://raw.githubusercontent.com/ArcQian-bit/codex-real-engineer-skills/main/install.sh | bash

Restart Codex after installing so the new skill is discovered.

Why It Exists

Many AI coding workflows fail because the assistant jumps straight to a plausible implementation. Real Engineer for Codex nudges Codex toward reproducible evidence, narrow changes, and explicit tradeoffs.
