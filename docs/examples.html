Real Engineer for Codex

Example Prompts

Positive Trigger: Diagnose

Use real-engineer to debug this failing checkout test.

Expected behavior: Codex should reproduce, isolate the smallest failing case, rank hypotheses, fix the proven cause, and keep a regression test.

Positive Trigger: TDD

Use real-engineer and implement this endpoint test-first.

Expected behavior: Codex should start with a failing test, implement the smallest passing change, then refactor.

Positive Trigger: Grill

Use real-engineer to stress-test this feature idea before coding.

Expected behavior: Codex should ask concrete product and engineering questions with recommended defaults.

Negative Trigger

Make this README sound more casual.

Expected behavior: the skill should not activate. This is writing/editing work, not debugging, TDD, prototyping, or architecture review.
